# LinguBoost

## A simple language-learning themed project built to practice JavaScript fundamentals and DOM manipulation.

# 🎯 Purpose

## This project was created as a JavaScript practice exercise to improve skills in:

- DOM Manipulation
- Event Handling
- Working with Forms
- Dynamic Content Rendering
- User Interaction Design

# ✨ Features:

- Interactive language learning interface
- Dynamic content updates
- User input handling
- Responsive layout

# 🛠️ Built With:
- HTML5
- CSS3
- JavaScript

# 📚 What I Practiced

## During this project I focused on:

- Selecting and manipulating DOM elements
- Handling user events
- Updating UI dynamically
- Creating reusable JavaScript functions
- Organizing code structure
- Improving responsiveness and user experience

## Key JavaScript Concepts Practiced:

- querySelector / querySelectorAll
- addEventListener
- Event Delegation
- Conditional Rendering
- Array Methods (forEach)

# 📁 Project Status

## This project was built for learning purposes and is not intended for production use.
